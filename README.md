# Dino Fichas

## Instrucciones para los Sonidos
Como soy una IA basada en texto, no puedo generar archivos de audio reales (.mp3). He creado la estructura para que funcionen, pero necesitas añadir tus propios archivos de sonido en la carpeta `assets/sounds/`:

1.  **correct.mp3**: Un sonido de "Ding" o "Acierto".
2.  **wrong.mp3**: Un sonido de "Error" o "Buzz".
3.  **win.mp3**: Una música de celebración o aplausos.

Puedes descargar sonidos gratuitos en sitios como [Freesound.org](https://freesound.org/).

## Cómo añadir más Dinosaurios
1.  Crea una imagen (JPG/PNG/SVG) y guárdala en `assets/images/`.
2.  Crea un archivo JSON en la carpeta `data/` (ej: `triceratops.json`) siguiendo el formato de los existentes.
3.  Añade el dinosaurio a la lista en `data/list.json`.
